# Mohtashim Javed | Professional Portfolio 🚀

Welcome to my professional portfolio! This is a high-performance, visually stunning web application built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**. It showcases my journey, skills, and projects as a Full Stack Developer specializing in the MERN stack and React Native.

![Portfolio Preview](https://api.dicebear.com/7.x/notionists/svg?seed=Mohtashim&backgroundColor=0b0f19)

## ✨ Key Features

- **Modern & Responsive UI**: A sleek, dark-themed interface built with a mobile-first approach.
- **Dynamic Project Showcase**: An interactive carousel to explore my latest work with live demos and source code links.
- **Advanced Animations**: Smooth transitions and micro-interactions powered by Framer Motion and GSAP.
- **Glassmorphism & 3D Effects**: Premium aesthetics with frosted glass effects, vibrant gradient accents, and 3D perspective transformations.
- **Typography**: Optimized for readability using **JetBrains Mono**, **Rubik**, and **Zen Dots** fonts.
- **SEO Optimized**: Built with Next.js best practices for performance and search engine visibility.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Language**: JavaScript / React

## 🚀 Getting Started

### Prerequisites

- **Node.js**: v18.0.0 or higher
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohtashimjaved/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   npm start
   ```

The application will be available at `http://localhost:3000`.
